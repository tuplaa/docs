# OpenAPI Documentation

Welcome to the NVP Platform API documentation. This section provides comprehensive documentation for all available APIs.

## Available APIs

### PAM (Player Account Management)
The PAM API provides endpoints for managing player accounts, authentication, and related operations. Key features include:
- Player registration and authentication
- Account management
- Profile updates
- Security features

[View PAM API Documentation](./pam)

### Backoffice
The Backoffice API provides administrative and management capabilities. Key features include:
- User management
- System configuration
- Reporting and analytics
- Audit logging

[View Backoffice API Documentation](./bo)

## Getting Started

To start using our APIs:

1. **Authentication**
   - All APIs require proper authentication
   - Use JWT tokens for API requests
   - Keep your credentials secure

2. **API Access**
   - Ensure you have the necessary permissions
   - Review the API documentation for your specific needs
   - Follow the integration guidelines

3. **Best Practices**
   - Implement proper error handling
   - Follow rate limiting guidelines
   - Use secure communication (HTTPS)
   - Keep your API client updated
